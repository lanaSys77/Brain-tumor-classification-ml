# 🧠 Brain Tumor Classification using Machine Learning

A comparative machine learning project for classifying brain MRI images into tumor categories using multiple models.

---

## 📌 Project Overview

This project focuses on classifying brain MRI images into three categories:

- Glioma Tumor
- Pituitary Tumor
- No Tumor

The main goal is to compare the performance of three different machine learning models:

- Naive Bayes
- Decision Tree
- Multi-Layer Perceptron (MLP)
